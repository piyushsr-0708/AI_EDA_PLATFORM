Project: AI_EDA_PLATFORM

Goal:
Build a Streamlit application that:

1. Accepts any CSV dataset
2. Profiles the dataset
3. Cleans the dataset automatically
4. Detects ML task type
5. Performs EDA
6. Trains ML models automatically
7. Selects best model
8. Forecasts if time-series
9. Saves reports
10. Saves trained models

Current Modules:
- app.py
- profiler.py
- cleaner.py
- task_detector.py

Rules:
- Keep code simple
- Use modular architecture
- Do not create unnecessary classes
- Prefer functions
- Every module should be independently testable
